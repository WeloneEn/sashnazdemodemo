# Babijon — дизайн-портфолио

Персональный сайт-портфолио для специалиста по плитке и дизайну санузлов.

## Локальный запуск

1. Откройте `index.html` в браузере.
2. Для корректной работы ссылок лучше запустить локальный сервер:

```bash
python -m http.server 8080
```

После этого откройте `http://localhost:8080/`.

## Что внутри

- Главная страница с презентацией и анимациями.
- Портфолио с интерактивной пандой и пасхалками.
- Блок "О Сашуле" с цветовой палитрой и фоновыми эффектами ремонта.
- Страница контактов с `mailto`-формой.

## Деплой на GitHub Pages

В репозитории есть workflow: `/.github/workflows/deploy.yml`.
Публикация выполняется при push в ветку `main`.
